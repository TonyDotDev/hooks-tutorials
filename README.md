# hooks-tutorials
Demonstration repo for hooks tutorial blog posts.

This contains demos of the following hooks:
- useOnClickOutside

## How To Install
```sh
git clone git@github.com:TonyDotDev/hooks-tutorials.git
cd hooks-tutorials
npm install
```

## How To Run
```sh
npm dev
```

Navigate to `http://127.0.0.1:5173` in your browser.
